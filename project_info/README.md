# Задача для веб- и fullstack - разработчиков от Digital агентства Победа

## **Список пользователей**

### **Цель**

Создать простое веб-приложение для отображения списка пользователей, получение информации о конкретном пользователе и добавление нового пользователя.

### **Требования**

**1. Бэкенд (Python + Flask):**

- Хранить в SQLite базе данных информацию о пользователях (id, имя, email).
- Иметь два эндпоинта:
  - /users (GET) - возвращает список всех пользователей в формате JSON
  - /users/\<id> (GET) - возвращает пользователя по id в формате JSON
- Добавить базовую обработку ошибок (например, если пользователя с запрошенным id не существует).

**2. Фронтенд (HTML/CSS/JavaScript):**

- Создать простую HTML страницу с использованием vanilla JS.
- Использовать Fetch API для получения данных с эндпоинта /users бэкенд приложения.
- Отобразить список пользователей в таблице.
- По клику на пользователя показывать детальную информацию о нем в модальном окне.
- Использовать Bootstrap для стилизации.

**3. Интеграция (Python + JavaScript):**

- Добавить на фронтендную страницу возможность добавления нового пользователя через форму.
- Форма отправляет данные на бэкенд через Fetch API.
- Обновлять список пользователей без перезагрузки страницы.
- Показывать сообщение об успешном добавлении или ошибке.

**4. База данных:**

- Использовать SQLite.
